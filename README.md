# 💱 Currency Converter Web App

A simple and responsive **Currency Converter** built using **HTML, CSS, and JavaScript**.
It fetches real-time exchange rates using an API and allows users to convert between currencies instantly.

---

## 🚀 Features

* 🌍 Convert between multiple currencies
* 🔄 Real-time exchange rates using API
* 🏳️ Dynamic country flags update
* 🔁 Swap currencies with one click
* ⚡ Instant calculation
* 📱 Responsive and clean UI

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript (ES6)** – Logic & API handling
* **ExchangeRate API** – Live currency data
* **Flags API** – Country flags

---

## 📂 Project Structure

```
Currency-Converter/
│── index.html
│── style.css
│── app.js
│── code.js   (country list)
```

---

## ⚙️ How It Works

1. User enters an amount
2. Selects "From" and "To" currencies
3. App fetches exchange rate from API
4. Displays converted amount instantly

---

## 🔗 API Used

* Exchange Rate API:
  https://v6.exchangerate-api.com/

* Flags API:
  https://flagsapi.com/

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start converting currencies 🎉
